# Apex Bank Management System

A premium, modern desktop-based Bank Management System built using **Python**, **Tkinter**, and **SQLite**. The application features a sleek dark-themed dashboard interface with advanced account management, transaction logs, data export options, and profile photo integration.

---

## Key Features

1. **Dashboard Command Center**
   - High-level metric cards showing Total Registered Accounts, Total Bank Deposits (USD), and Total Transaction Volume.
   - A real-time recent transactions ledger list showing chronological activities.
   - Visual account portfolio representation (Savings vs. Current distribution) using custom Tkinter Canvas bar graphs.
   
2. **Account Central**
   - **Account Registry**: Fully searchable table listing all registered accounts with quick filter actions.
   - **Detailed Profile Cards**: Selection displays full customer data (Account Number, Holder Name, IFSC Code, Address, Account Balance, Joined Timestamp, Status) along with a cropped/resized profile picture.
   - **Modify Profile**: Directly modify customer names, addresses, IFSC codes, status (Active, Suspended, Closed), or update their profile photos.
   - **Individual Statements**: Export client statements with personal details and complete history directly to a CSV file.
   - **Account Registration**: Self-contained client sign-up form with optional profile image upload and random Account Number auto-generator.

3. **Transaction Simulator**
   - **Cash Deposit**: Deposit funds to any active account.
   - **Cash Withdrawal**: Withdraw funds from any active account (enforces database balance checks).
   - **Account Transfer**: Transfer funds between two active client accounts with live integrity verification.
   - **Live Account Verification**: Real-time DB lookup prints account owner's status and current balance dynamically on keypress to prevent erroneous inputs.

4. **Data Management Center**
   - Save full system databases to independent CSV spreadsheet folders.
   - Quick exports of **Accounts Ledger** and **Transaction Ledger** via file selector dialogs.

## Requirements & Setup

### Prerequisites
Ensure you have **Python 3.8+** installed. The only external dependency is **Pillow (PIL)** for image processing and cropping.

### Installation
Open your terminal in the project directory and install the required library:
```bash
pip install Pillow
```

### Execution
Run the system by invoking `main.py` with Python:
```bash
python main.py
```

## Visual Design Details

- **Deep Navy Base Background (`#0f172a`)**: Immersive dark mode to look extremely premium.
- **Card Styling (`#1e293b`)**: Clear box layout dividers with slate borders.
- **Custom Canvas Components**: Custom shapes with anti-aliasing to draw smooth rounded buttons (`ModernButton`) and graphs without raw system style overrides.
- **Color Accent Guides**: 
  - Indigo Accent (`#6366f1`): Primary actions.
  - Emerald Accent (`#10b981`): Balanced deposits and success notifications.
  - Rose Accent (`#f43f5e`): High-importance withdrawals and alerts.
